# AI Notes Generator

An AI-powered web application that generates concise, well-structured study notes for any topic using **Google Gemini AI**. Built with **React (frontend)** and **Flask (backend)** and deployed on cloud platforms.

---

## Live Demo

1.Frontend: https://ai-notes-genrator.vercel.app/
2.Backend: https://ai-notes-genrator.onrender.com/ 

---

## Features

- AI-generated study notes using Gemini API  
- Fast response backend (Flask API)  
- Clean and responsive UI (React)  
- Fully deployed web application  
- Works on mobile & desktop  

---

## Tech Stack

### Frontend:
- React (Vite)
- HTML, CSS, JavaScript

### Backend:
- Python
- Flask
- Flask-CORS
- Gunicorn

### AI Model:
- Google Gemini API (`google-genai`)

### Deployment:
- Frontend: Vercel  
- Backend: Render
